#### 华为
演示设备：华为畅享20、EMUI9.1、安卓9 。
一、首先打开手机设置，找到系统点击打开。

二、找到开发人员选项，点击打开（如果没有，则进入关于手机中，连续点击7次系统版本号即可出现开发人员选项）。

三、在该界面中找到指针位置功能，开启该功能，效果如下图，开启该功能之后，手指只要是触碰到手机屏幕上任意一点，就会有横竖两条定位线。
